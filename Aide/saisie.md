# Saisie de transaction

## Sélectionner un pilote

Utilisez la liste déroulante en haut de page pour sélectionner le pilote concerné, puis cliquez sur **Afficher**.

## Saisie Rapide

Permet d'enregistrer rapidement une transaction prédéfinie :

1. Choisissez le **type** (Encaissement / Vente)
2. Sélectionnez la **catégorie** dans la liste
3. Saisissez le **montant** (pré-rempli si un tarif est configuré)
4. Cliquez sur **Ajouter**

## Saisie Complète

Permet de saisir une transaction libre avec un intitulé personnalisé.

## Enregistrer un vol

Le bouton **Enregistrer un vol** (en-tête) ouvre un formulaire pour saisir directement un vol depuis cette page.

## Export

- **Export PDF** — génère l'extrait de compte du pilote
- **Facture** — génère une facture par année
